# Round Badges — 9 Topic Emblems

**For:** Small circular emblem icons, one per quiz round topic, shown on the round-select screen and round intros.

**Aspect ratio / size:** 1:1 square, generate as a 3x3 grid sheet at 2048x2048 (or generate each badge individually at 512x512 using the same prompt with only its badge line). Each badge must read at ~64px.

## Prompt

```
hand-inked retro-futurist mascot robot art, thick cream sticker-style outlines, subtle paper grain and ink texture, warm palette — ink #161412, cream #f5efe0, dim paper #d8d2c2, orange #e87722, bright orange #ff9a3c, beige #b9b1a0, danger red #c0392b — CRT-screen faces with glowing features, soft neutral gray studio background unless the prompt says otherwise, no photorealism, no text/lettering unless the prompt explicitly asks for it.

a set of nine small circular emblem badge icons arranged in a neat 3x3 grid, flat sticker style, each badge a cream disc with a thick ink outer ring, a single bold orange-and-ink symbol centered inside, minimal shading, identical framing and stroke weight across all nine, evenly spaced on a soft neutral gray background: 1) a wide rounded rectangle frame with a smaller glowing rectangle inside it, a context window; 2) a circular arrow loop with a tiny robot-head node on the ring, an agentic loop; 3) an unrolled paper scroll with faint squiggle lines, a prompt scroll; 4) a funnel with three dots entering the top and one bright orange dot leaving the bottom, curation; 5) a file card with a folded corner and a small glowing memory dot, a memory file; 6) an electric plug whose prongs spark tiny orange ticks, tools; 7) a magnifying glass with a bold orange check mark inside the lens, verification; 8) a rounded shield with a single watchful eye emblem, review; 9) a stubby retro rocket with a small orange flame and a tiny circular co-pilot window, launch; crisp, geometric, instantly readable at tiny sizes, matched as a family
```

## Variations

- "Locked" state set: same nine badges in desaturated beige with a small ink padlock over each symbol.
- "Completed" state set: orange ring glow plus a tiny starburst notch at the top of each ring.
- Hex-badge alternate: identical symbols on rounded hexagon plates instead of circles for a more arcade-token feel.
