# Beyond Boring: Battle of the Backlog — Image Prompts

Each `.md` file in this folder contains one fully self-contained image-generation prompt: open the file, copy the entire fenced code block (boilerplate and asset description together — never trim the boilerplate, it carries the shared style and palette), paste it whole into your image generator, and generate at the aspect ratio / size listed at the top of that file. Keep the palette exactly as specified (ink `#161412`, cream `#f5efe0`, dim paper `#d8d2c2`, orange `#e87722`, bright orange `#ff9a3c`, beige `#b9b1a0`, danger red `#c0392b`) and reject any result that drifts photorealistic, loses the thick cream sticker outlines, or invents text — only `title-logo` is allowed lettering. Compare candidates against the reference art one folder up (`../brilliant.png`, `../boring.png`) before accepting, and use each file's "Variations" section for alternate takes or state swaps.
